Grid vs Flexbox

La diferencia fundamental.

/* Flexbox (1D): */
/* Una fila O una columna. El contenido dicta el diseño. */
/* "Tengo estos ítems, ¿cómo los acomodo?" */

/* Grid (2D): */
/* Filas Y columnas simultáneas. El diseño dicta el contenido. */
/* "Tengo este plano arquitectónico, pon los ítems aquí." */

Terminología Básica

Es vital entender estos términos:

1. Grid Container: El padre (display: grid).
2. Grid Item: Los hijos directos.
3. Grid Line: Las líneas divisorias (1, 2, 3...).
4. Grid Track: El espacio entre dos líneas (fila/columna).
5. Grid Area: Un rectángulo de 1 o más celdas.
6. Grid Cell: La unidad mínima (intersección fila/col).

Definiendo la Cuadrícula

Activamos el grid y definimos el tamaño de las pistas.

.contenedor {
  display: grid;
  
  /* Tres columnas: 200px, 150px y 200px */
  grid-template-columns: 200px 150px 200px;
  
  /* Dos filas: 100px y 300px */
  grid-template-rows: 100px 300px;
}
4 / 21

La Unidad `fr` (Fracción)

Una unidad exclusiva de Grid. Representa una fracción del espacio disponible.

.contenedor {
  display: grid;
  /* 3 columnas iguales */
  grid-template-columns: 1fr 1fr 1fr;
}

.mixto {
  /* 100px fijos, y el resto se divide 2 a 1 */
  grid-template-columns: 100px 2fr 1fr;
}
5 / 21

Función repeat()

Evita escribir 1fr 1fr 1fr 1fr.... Simplifica la repetición de patrones.

.galeria {
  /* Crea 12 columnas de 1fr cada una */
  grid-template-columns: repeat(12, 1fr);
}

.patron {
  /* Repite el patrón "100px 50px" cuatro veces */
  grid-template-columns: repeat(4, 100px 50px);
}
6 / 21

Función minmax()

Define un rango de tamaño. "Sé al menos así de grande, pero no más grande que esto".

.sidebar {
  /* Mínimo 200px.
     Máximo 1fr (lo que sobre). */
  grid-template-columns: minmax(200px, 1fr) 3fr;
}

/* Útil para que el contenido no colapse más allá 
   de lo legible. */
7 / 21

Gap (Espaciado)

Espacio entre celdas (gutters). Funciona igual que en Flexbox.

.grid {
  /* Fila y Columna */
  gap: 20px;

  /* row-gap | column-gap */
  gap: 10px 50px;
}
/* Nota: Antiguamente era 'grid-gap', 
   pero ahora es solo 'gap'. */
8 / 21

Posicionamiento por Líneas (Hijos)

Para colocar un ítem, le decimos en qué línea empieza y en cuál termina.

.item-grande {
  /* Empieza en la línea de columna 1 */
  grid-column-start: 1;
  /* Termina en la línea de columna 3 (ocupa 2 celdas) */
  grid-column-end: 3;

  /* Shorthand: start / end */
  grid-row: 1 / 4; 
}
9 / 21

Palabra Clave: span

En lugar de calcular el número de línea final, decimos "ocupa X celdas".

.header {
  /* Empieza en la línea 1 y abarca 4 columnas */
  grid-column: 1 / span 4;
}

.sidebar {
  /* Ocupa 2 filas hacia abajo */
  grid-row: span 2;
}

Grid Template Areas (Muy Visual)

Puedes dibujar tu layout usando nombres y "arte ASCII" en CSS. Es increíblemente legible.

.layout {
  display: grid;
  grid-template-columns: 200px 1fr;
  
  grid-template-areas: 
    "cabecera cabecera"
    "menu     contenido"
    "pie      pie";
}
11 / 21

Asignando Áreas (Hijos)

Una vez definido el mapa en el padre, asignas los hijos a sus nombres.

.header  { grid-area: cabecera; }
.sidebar { grid-area: menu; }
.main    { grid-area: contenido; }
.footer  { grid-area: pie; }

/* Si cambias el mapa en el padre, 
   los elementos se mueven solos. */

Justify & Align Items

Controla la alineación del contenido DENTRO de su celda de grid.

.contenedor {
  /* Horizontal (start, end, center, stretch) */
  justify-items: center;

  /* Vertical (start, end, center, stretch) */
  align-items: center;
  
  /* Shorthand: align | justify */
  place-items: center center;
}
13 / 21

Justify & Align Content

Si la cuadrícula total es más pequeña que el contenedor, ¿dónde colocamos la cuadrícula entera?

.contenedor {
  height: 100vh;
  display: grid;
  
  /* Mueve TODA la cuadrícula al centro */
  align-content: center;
  justify-content: center;
}

Grid Implícito

Si defines 2 filas pero añades 10 ítems, Grid crea filas automáticamente. Esto es el Grid Implícito.

.contenedor {
  grid-template-rows: 100px 100px; /* Explícito */
  
  /* Define el tamaño de las filas auto-creadas */
  grid-auto-rows: 200px; 
  
  /* O usa minmax para que sean flexibles */
  grid-auto-rows: minmax(100px, auto);
}
15 / 21

Grid Auto Flow

Controla cómo el algoritmo de auto-colocación rellena el grid.

.contenedor {
  /* Defecto: rellena filas */
  grid-auto-flow: row; 

  /* Rellena columnas (añade columnas implícitas) */
  grid-auto-flow: column;

  /* Intenta llenar los huecos vacíos con 
     ítems pequeños más adelante en el HTML */
  grid-auto-flow: dense; 
}
16 / 21

Magia Responsiva: auto-fit

El truco más famoso de Grid. Columnas responsivas sin Media Queries.

.tarjetas {
  display: grid;
  grid-template-columns: 
    repeat(auto-fit, minmax(250px, 1fr));
}

/* Traducción: "Crea tantas columnas de 250px como quepan.
   Si sobra espacio (1fr), estíralas para llenar." */
17 / 21

auto-fit vs auto-fill

La diferencia sutil pero importante cuando hay poco contenido.

/* auto-fit (Ajustar): */
/* Si tengo 1 ítem y caben 3, estira el ítem 
   para ocupar TODO el ancho. (Colapsa celdas vacías). */

/* auto-fill (Rellenar): */
/* Si tengo 1 ítem y caben 3, deja el espacio 
   de los otros 2 vacíos. (Mantiene estructura). */
18 / 21

Superposiciones (Layering)

A diferencia de Flexbox, en Grid las celdas pueden compartir el mismo espacio. Usa z-index para ordenar.

.foto {
  grid-column: 1 / 3;
  grid-row: 1 / 3;
}

.titulo-encima {
  grid-column: 1 / 3;
  grid-row: 2;
  z-index: 10;
  align-self: center;
}

Grid DevTools

Trabajar con Grid sin inspeccionar es difícil. Usa las herramientas de desarrollo.

2. Busca el elemento con 'display: grid'.
3. Haz clic en la etiqueta 'grid' pequeña.
4. ¡Verás las líneas y números superpuestos en pantalla!

Resumen y Mejores Prácticas

Grid es para el "Macro Layout", Flexbox para el "Micro Layout".

1. Usa Grid para la estructura principal (Header, Main, Footer).
2. Usa 'grid-template-areas' para hacer el código legible.
3. Usa 'gap' generosamente.
4. Usa 'repeat(auto-fit, minmax(...))' para cards responsivas.
5. No temas mezclar: Un ítem de Grid puede ser contenedor Flex.
